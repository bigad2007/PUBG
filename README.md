## 🇰🇷 한국어 안내

### 이 페이지가 무엇인가요?

이 페이지는 배틀그라운드(PUBG)를 처음 접하는 플레이어 또는 저희 서버에 새로 참여하는 분들을 위한 **맵 교육 가이드**입니다.  
에란겔부터 론도까지, PUBG의 모든 맵에 대한 특징, 크기, 주요 지역, 전투 스타일을 정리했습니다.

### 포함된 내용

| 항목 | 설명 |
|---|---|
| 🎮 배그 기초 | 블루존, 레드존, 케어패키지, 핫드롭 개념 |
| 🗺️ 전체 맵 목록 | 9개 맵 카드 (에란겔, 미라마, 산호크, 비켄디, 태이고, 데스톤, 카라킨, 론도, 파라모) |
| 📋 맵 상세 분석 | 크기, 지형, 특수 시스템, 핫드롭/세이프드롭 위치 |
| ⚠️ 저작권 안내 | PUBG 공식 에셋 및 이용약관 안내 |

### 맵 이미지 출처

맵 이미지는 PUBG 공식 GitHub 저장소 [`pubg/api-assets`](https://github.com/pubg/api-assets)의 공개 에셋을 사용합니다.  
사용 시 [PUBG Developer Terms of Service](https://developer.pubg.com/tos)를 준수해야 합니다.

## 🇬🇧 English Guide

### What is this page?

This is an **educational map guide** for new players or those joining our server for the first time.  
It covers all PUBG maps from Erangel to Rondo — including terrain features, map sizes, key locations, and combat styles.

### What's included

| Section | Description |
|---|---|
| 🎮 PUBG Basics | Blue Zone, Red Zone, Care Package, Hot Drop concepts |
| 🗺️ All Maps Overview | 9 map cards (Erangel, Miramar, Sanhok, Vikendi, Taego, Deston, Karakin, Rondo, Paramo) |
| 📋 Detailed Map Analysis | Size, terrain, special systems, hot drop & safe drop locations |
| ⚠️ Copyright Notice | PUBG official asset & ToS information |
